# HandasaimSchedule
Client for the Handasaim School schedule. Using Github Actions as the backend and Svelte frontend.


# Development - 
run front end - 
```
cd frontend
npm install
npm run dev
```
start serverless function localy-
```
cd backend
python3 -m venv env
source env/bin/activate
pip3 install -r requirments
python3 main.py
```
